# 📈 Stock Price Predictor

A beginner-friendly Machine Learning project that predicts stock closing prices using Linear Regression.

## Features

- Download stock data using Yahoo Finance
- Data preprocessing using Pandas
- Linear Regression Model
- Accuracy Evaluation
- Visualization of Predicted vs Actual Prices

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn
- yFinance

## Installation

```bash
pip install -r requirements.txt
```

## Run Project

```bash
python stock_predictor.py
```
